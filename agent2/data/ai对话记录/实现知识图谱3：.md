# 实现知识图谱3（已核对现有实现）

## 目标需求
1. 对于作者查询，默认给出作者所有论文；用户点击作者后展示详细信息（所属机构、奖项、团队等）。  
2. 对于论文查询，默认给出作者；用户点击作者后展示作者详细信息（与作者查询一致）。  
3. 作者之间支持相似关系，默认隐藏；点击节点后按开关展示相似作者，再点击可查看相关作者信息。  
4. 论文之间支持相似关系，默认隐藏；点击节点后按开关展示相似论文，再点击可查看相关论文信息。  
5. 交互声明需在图谱空白区域明确展示。  

## 现有实现核对结论
- 已完成需求 1：后端作者节点保留 `papers/paper_count`，点击作者可在右侧查看论文列表、机构、省奖、国奖、团队。  
- 已完成需求 2：论文查询默认返回作者-论文关系，点击作者可查看与作者查询一致的完整详情。  
- 已完成需求 3：前端提供“相似作者”开关，默认关闭；点击作者后按需请求并展示同类型相似作者。  
- 已完成需求 4：前端提供“相似论文”开关，默认关闭；点击论文后按需请求并展示同类型相似论文。  
- 已完成需求 5：图谱画布顶部已有声明文案，说明默认关系与相似开关交互。  

## 对应实现位置
- 后端查询与相似逻辑：`src/kg/simple_engine.py`  
- 后端作者扩展信息聚合：`src/kg/simple_repository.py`  
- 图谱 API：`src/api/main.py`（`/api/kg/query`、`/api/kg/subgraph`）

## 快速验收步骤
1. 打开前端 `/knowledge-graph`，输入作者名查询：应看到作者及其论文关系。  
2. 点击作者节点：右侧应显示机构、论文列表、省奖、国奖、团队。  
3. 开启“相似作者”，再点击作者：图中应出现相似作者虚线关系。  
4. 输入论文名查询并点击论文：应看到作者关系；开启“相似论文”后再点击论文，应出现相似论文虚线关系。  
5. 观察画布顶部声明文案：应明确“默认隐藏相似，开关控制展示”。
